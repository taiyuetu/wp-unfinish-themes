Project Overview

Northline House needed a digital presence that felt premium without becoming ornate. We reduced visual noise, clarified the booking path, and built a measured interface that lets the imagery and typography do the work.

Client
Northline House
Scope
Brand direction, UX, interface design, build
Role
Creative direction and implementation

The layout was designed around one clear hierarchy: the property, the offer, and the booking action. We used slower transitions, tighter text measures, and a more editorial rhythm so the page felt composed rather than crowded.

The final result made the booking journey easier to understand and the brand more memorable, which is the only combination that matters for a portfolio piece of this kind.

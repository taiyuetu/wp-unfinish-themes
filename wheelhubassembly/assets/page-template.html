Overview

This is a default page template featuring a page banner, sidebar navigation, and main content area. Use this as a starting point for creating new pages such as capabilities, manufacturing, quality assurance, or sustainability pages.

The layout follows the established design system with consistent spacing, typography, and component patterns. The sidebar provides navigation within the page or section, while the main content area supports rich content including headings, paragraphs, lists, images, and more.

Content Sections

Each content section can be styled as a block with consistent padding and margins. The modular approach allows you to create complex pages by combining different content types.

Features Include
